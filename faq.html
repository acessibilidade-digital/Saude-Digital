Perguntas Frequentes (FAQ)

O que é o projeto Saúde Digital?

É uma iniciativa que visa facilitar o acesso à saúde digital para pessoas com dificuldades no uso de celulares, oferecendo lembretes de consultas e integração com o WhatsApp.

Como receberei os lembretes de consulta?

Os lembretes serão enviados diretamente pelo WhatsApp, utilizando linguagem simples e recursos visuais e sonoros para melhor compreensão.

Preciso instalar algum aplicativo adicional?

Não. Todo o sistema é acessado via navegador, sem a necessidade de instalar aplicativos extras.

O sistema é gratuito?

Sim, o acesso ao sistema é totalmente gratuito para os usuários.

Quem pode utilizar o sistema?

Qualquer pessoa que tenha dificuldade em utilizar tecnologias digitais para agendamento e lembretes de consultas médicas.
