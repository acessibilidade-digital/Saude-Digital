Solução Proposta

O projeto propõe uma plataforma web com foco em simplicidade e acessibilidade. Nosso objetivo é facilitar a confirmação de consultas e o envio de lembretes para pacientes que têm dificuldade com tecnologias digitais.

Principais Funcionalidades

  • Envio de lembretes automáticos via WhatsApp
  • Interface limpa e intuitiva
  • Textos com linguagem simples
  • Recursos de áudio e contraste para pessoas com deficiência visual
  • Navegação adaptada para idosos

Impacto Esperado

Com essas ferramentas, esperamos reduzir o absenteísmo de 20% para menos de 10%, contribuindo para uma saúde pública mais eficiente e inclusiva.
